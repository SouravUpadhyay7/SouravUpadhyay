# Premium Portfolio v2026

A hyper-modern, minimal, and futuristic personal portfolio designed for Sourav Upadhyay.

## Features
- **Ultra-Modern UI**: Glassmorphism, floating bottom navbar, and smooth animations.
- **Micro-interactions**: Hover effects, scroll reveals, and 3D card parallax.
- **Dynamic Theming**: Light/Dark mode with system persistence.
- **SEO Optimized**: Semantic HTML and meta tags included.
- **Modular Structure**: Seperate HTML files for easier content management.

## Project Structure
- `/index.html`: Home page with animated hero.
- `/about.html`: Detailed bio and skill grid.
- `/projects.html`: Featured work with hover-reveal details.
- `/certifications.html`: Professional achievements grid.
- `/contact.html`: Interactive contact form.
- `/css`: Styling modules (`themes`, `style`, `animations`).
- `/js`: Logic modules (`theme`, `animations`, `main`).
- `/assets`: Images, icons, and certificates.

## Customization
- **Colors**: Edit `css/themes.css` to change the accent gradients.
- **Content**: Each section is clearly commented in its respective HTML file.
- **Images**: Replace files in `assets/images/` to update your profile and project shots.
